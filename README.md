# Abnib Glamping 2026

Repo for a disposable website to coordinate [Abnib](https://abnib.co.uk/) folks as they plan a Summer 2026 glamping trip.

## Developing

Requires Ruby; tested with version 3.3.10. Run `bundle` to get dependencies.

Run `middleman server` to develop locally; `middleman build` to make a production build.

Key files you're looking for:
- `source/index.html.erb` - the homepage content
- `source/layouts/layout.erb` - the wrapper with header and footer details
- `source/stylesheets/site.css.scss` - stylesheet for the site
